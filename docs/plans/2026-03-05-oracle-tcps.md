# Oracle TCPS Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Encrypt both legs of Oracle communication — client→proxy (TCPS) and proxy→Oracle (TCPS) — using the project's existing self-signed CA.

**Architecture:** The proxy wraps every accepted Oracle connection with `tls.Server` using the existing `*tls.Config` already passed to `HandleClient`. For the proxy→Oracle direction, `oracle.Handler` gains an `OracleTLS *tls.Config` field; `ConnectAndAuth` uses `tls.Dial` for the relay socket and injects `SSL=TRUE` into the go-ora ping URL. Oracle is configured for TCPS via an init script that populates an Oracle Wallet from a PKCS12 bundle generated by the existing cert script.

**Tech Stack:** Go `crypto/tls`, `github.com/sijms/go-ora/v2`, Oracle Wallet (`orapki`), `gvenzl/oracle-free:23-slim`, openssl (PKCS12 export).

### Task 1: Add Oracle TLS certificates to the cert generation script

**Files:**
- Modify: `certs/generate.sh`

**Background:**
Oracle needs a server certificate for TCPS. We generate `oracle-server.key`/`oracle-server.crt` (signed by the project CA, SAN=`DNS:oracle`) and `oracle.p12` (PKCS12 bundle with no passphrase) for the Oracle Wallet.

**Step 1: Edit `certs/generate.sh` — insert before the "Cleanup CSRs" line**

Find the block that ends with:
```bash
# Vault TLS dir (VAULT_DEV_TLS_CERT_DIR expects tls.crt and tls.key)
VAULT_TLS="$OUT/vault-tls"
mkdir -p "$VAULT_TLS"
cp "$OUT/vault-server.crt" "$VAULT_TLS/tls.crt"
cp "$OUT/vault-server.key" "$VAULT_TLS/tls.key"
```

Insert this block immediately after it (before `# Cleanup CSRs`):
```bash
# Oracle TCPS cert (SAN=DNS:oracle for proxy→oracle TLS verification)
openssl genrsa -out "$OUT/oracle-server.key" 2048
openssl req -new -key "$OUT/oracle-server.key" -out "$OUT/oracle-server.csr" \
  -subj "/CN=oracle"
openssl x509 -req -in "$OUT/oracle-server.csr" -CA "$OUT/ca.crt" -CAkey "$OUT/ca.key" \
  -CAcreateserial -out "$OUT/oracle-server.crt" -days 365 \
  -extfile <(printf "subjectAltName=DNS:oracle")

# Oracle Wallet PKCS12 bundle (no passphrase — orapki import_pkcs12 uses -pkcs12pwd "")
openssl pkcs12 -export \
  -in "$OUT/oracle-server.crt" \
  -inkey "$OUT/oracle-server.key" \
  -certfile "$OUT/ca.crt" \
  -out "$OUT/oracle.p12" \
  -passout pass:
```

**Step 2: Regenerate all certs**

```bash
cd certs && bash generate.sh
```

**Step 3: Verify oracle.p12 is a valid PKCS12**

### Task 2: Create Oracle TCPS Docker configuration files

**Files:**
- Create: `oracle/tcps/sqlnet.ora`
- Create: `oracle/tcps/listener.ora`
- Create: `oracle/tcps/setup-tcps.sh`

**Background:**
`gvenzl/oracle-free` runs scripts from `/container-entrypoint-initdb.d/` after the database initializes. The script populates an Oracle Wallet from `oracle.p12`, writes Oracle network config files, and reloads the listener to add a TCPS endpoint on port 2484.

**Step 1: Create `oracle/tcps/sqlnet.ora`**

```

SSL_CLIENT_AUTHENTICATION=FALSE
WALLET_LOCATION=(SOURCE=(METHOD=FILE)(METHOD_DATA=(DIRECTORY=/opt/oracle/wallet)))
```

**Step 2: Create `oracle/tcps/listener.ora`**

```
LISTENER =
  (DESCRIPTION_LIST =
    (DESCRIPTION =
      (ADDRESS = (PROTOCOL=TCPS)(HOST=0.0.0.0)(PORT=2484))
    )
  )

SSL_CLIENT_AUTHENTICATION=FALSE
WALLET_LOCATION=(SOURCE=(METHOD=FILE)(METHOD_DATA=(DIRECTORY=/opt/oracle/wallet)))
```

**Step 3: Create `oracle/tcps/setup-tcps.sh`**

```sh
#!/bin/bash
set -euo pipefail

mkdir -p /opt/oracle/wallet

# Create auto-login wallet and import the PKCS12 bundle
orapki wallet create -wallet /opt/oracle/wallet -auto_login_only
orapki wallet import_pkcs12 \
  -wallet /opt/oracle/wallet \
  -auto_login_only \
  -pkcs12file /tcps-certs/oracle.p12 \
  -pkcs12pwd ""

# Deploy Oracle network config
cp /tcps-config/sqlnet.ora /opt/oracle/oradata/tns/sqlnet.ora
cp /tcps-config/listener.ora /opt/oracle/oradata/tns/listener.ora

# Reload listener to pick up TCPS address
lsnrctl reload
```

**Step 4: Make the script executable**

```bash

```

### Task 5: Wrap client→proxy connection in TLS in oracle/handler.go

**Files:**
- Modify: `oracle/handler.go`

**Background:**
Oracle TCPS means TLS is established immediately after TCP connect, before any TNS. `HandleClient` receives the raw `net.Conn`; it must wrap it with `tls.Server` and complete the handshake before reading the TNS CONNECT packet.

**Step 1: Update `HandleClient` to use the TLS config**

Current:
```go
func (h *Handler) HandleClient(clientConn net.Conn, _ *tls.Config) (io.ReadWriteCloser, string, error) {
	// Oracle clients connect with plain TNS (no TLS/SSLRequest)
	pkt, err := readTNSPacket(clientConn)
	if err != nil {
		return nil, "", fmt.Errorf("reading TNS packet: %w", err)
	}
	...
	return &clientConn_{
		Conn:           clientConn,
		connectPayload: pkt.payload,
	}, serviceName, nil
}
```

Replace with:
```go
func (h *Handler) HandleClient(clientConn net.Conn, tlsConfig *tls.Config) (io.ReadWriteCloser, string, error) {
	// Oracle TCPS: TLS handshake happens immediately after TCP connect, before any TNS.
	tlsConn := tls.Server(clientConn, tlsConfig)
	if err := tlsConn.Handshake(); err != nil {
		return nil, "", fmt.Errorf("TLS handshake with oracle client: %w", err)
	}

	pkt, err := readTNSPacket(tlsConn)
	if err != nil {
		return nil, "", fmt.Errorf("reading TNS packet: %w", err)
	}

	if pkt.packetType != tnsConnect {
		return nil, "", fmt.Errorf("expected TNS CONNECT (type %d), got type %d", tnsConnect, pkt.packetType)
	}

	serviceName := parseServiceName(pkt.payload)
	if serviceName == "" {
		return nil, "", fmt.Errorf("could not extract SERVICE_NAME from CONNECT packet")
	}

	log.Printf("oracle client requests service: %s", serviceName)

	return &clientConn_{
		Conn:           tlsConn,
		connectPayload: pkt.payload,
	}, serviceName, nil
}
```

Key change: read from `tlsConn` (not `clientConn`), store `tlsConn` in `clientConn_.Conn` so `AcceptClient` writes back through TLS.

**Step 2: Add `buildOracleTLSConfig` helper at the bottom of `main.go`**

```go
func buildOracleTLSConfig(caPath string) (*tls.Config, error) {
	caPEM, err := os.ReadFile(caPath)
	if err != nil {
		return nil, fmt.Errorf("reading CA cert %s: %w", caPath, err)
	}
	pool := x509.NewCertPool()
	if !pool.AppendCertsFromPEM(caPEM) {
		return nil, fmt.Errorf("failed to parse CA cert")
	}
	return &tls.Config{
		RootCAs:    pool,
		ServerName: "oracle",
	}, nil
}
```
